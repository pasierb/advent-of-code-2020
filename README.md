# Advent of code 2020

https://adventofcode.com/2020 solutions

## Usage

### Get solution

```bash
# get most recent day solution
npm run solution

# or specify day
npm run solution day02
```

### Boostrap another day

```bash
npm run bootstrap <DAY>

# for example
npm run bootstrap day02
```

### Testing

```bash
# runs only most recent day tests
npm test
```
